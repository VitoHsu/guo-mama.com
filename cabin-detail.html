郭媽媽小木屋是一片由香杉木打造的獨棟小木屋社區，為每一個想慢下來、放鬆身心的旅人，留下一個不被打擾的安心空間。
我們也提供迎賓下午茶、元氣早餐、烤肉代辦與桶仔雞等選項，讓你不用奔波，也能在山中享受美好食光。
這裡不只是住一晚的地方，而是你放慢腳步、和家人朋友好好生活的空間。
郭媽媽小木屋，歡迎你來森林裡，住一個會留下記憶的夜晚。

  • Features 01獨棟小木屋
    高品質設備

    每棟香杉木打造的六人獨棟木屋，配有三張加大雙人獨立筒床與TOTO高級衛浴，提供如家般安心的住宿體驗。設有免費WIFI、廚房鍋具、吹風機與浴巾用品，生活細節一次備齊，拎包即可入住。

  • Features 02全方位休閒設施
    烤肉聚會體驗

    從電動麻將桌、卡拉OK（開放至22:00），到庭園大草皮、烤肉區雙爐烤架應有盡有，無論想放鬆、聊天、歡聚都能滿足。還可加購代辦烤肉餐（最低$1500起），讓旅行輕鬆又好玩。

  • Features 03迎賓下午茶
    元氣早餐，美食不間斷

    入住即享甜點、現煮咖啡、奶茶、桂花茶等迎賓飲品，隔天再送上三明治、水果、南瓜湯與手沖壺飲的元氣早餐，每一餐都是真誠款待。
